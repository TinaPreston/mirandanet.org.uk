Audio Guides

Several pages in this project have a loudspeaker link which takes you to the foot of the page where there should be an audio control allowing you to listen to an audio guide to the page. If you do not see the control, you are probably using an old browser, or need to download a plugin.

There is no audio guide for this page, but there is a very good introduction to Mindmapping on the Mindmapping page.
